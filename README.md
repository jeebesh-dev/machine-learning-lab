# Machine Learning Lab

Machine Learning laboratory assignments and practical implementations developed as part of the M.Sc. Computer Science curriculum.

## Experiments

* Find-S Algorithm
* Candidate Elimination Algorithm
* Decision Tree Learning
* Naive Bayes Classifier
* K-Nearest Neighbors (KNN)
* K-Means Clustering
* Linear Regression
* Artificial Neural Networks (ANN)
* Gradient Descent for Logic Gates

## Datasets

* Iris Dataset
* Diabetes Dataset

## Technologies Used

* Python
* Jupyter Notebook
* NumPy
* Pandas
* Scikit-learn
* Matplotlib
